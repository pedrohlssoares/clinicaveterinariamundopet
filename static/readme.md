# Pasta: Static

Esta pasta armazena todos os recursos estáticos e públicos que dão suporte visual e interativo às telas do sistema. Tudo o que está aqui dentro é acessado diretamente pelo navegador do cliente.

### O que deve ficar aqui?
* Subpasta `css/`: Para folhas de estilo que definem o design e o layout.
* Subpasta `js/`: Para scripts JavaScript que trazem interatividade ao front-end.
* Subpasta `images/`: Para logotipos, ícones e mídias estáticas do sistema.

### Regra de Ouro:
Nenhum arquivo nesta pasta deve conter código PHP ou lógica de servidor. São arquivos puramente estáticos entregues diretamente ao navegador.
